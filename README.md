# Advanced Combat System (Roblox Luau)

This project demonstrates an intermediate-to-advanced Roblox Luau combat system created for scripting skill verification.

## Features

* Metatable-based OOP combat system
* CFrame dash movement
* Raycast hit detection
* Physics knockback
* Combo attack system
* Cooldown management
* Server-side validation / anti-spam logic
* Roblox API usage and optimization

## Setup

1. Place the main Script inside `ServerScriptService`
2. Create a `RemoteEvent` named `CombatRemote` inside `ReplicatedStorage`
3. Create a `Tool` named `CombatTool`
4. Add the LocalScript used for attack and dash input
5. Publish and test in Roblox Studio
